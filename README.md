# Chat-bot facebook
> Send messages facebook when you offline

1. Run `npm install `

2. Edit file `login.js`
![Open file](https://i.imgur.com/QxJNrWy.png)

3.   Edit file `bot.js`
![Open file](https://i.imgur.com/zsyRrVq.png)

4. On cmd screen  
 - Run `node login.js` -> Created file ` appstate.json`
 - Run `node bot.js`
